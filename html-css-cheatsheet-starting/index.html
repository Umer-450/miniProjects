Swifty's Web Dev Cheatsheet

The Table Element

Tag Element Name Description
<table> Table The parent element of an HTML table
<thead>> Table Head The header row parent to the individual column heading elements
<th> Table Head The column heading name as part of the heading row
<tbody> Table Body The parent element containing all table rows
<td> Table Data The data within cells in the table
<tfoot> The footer row at the bottom of the page that can contain other table elements

Table Attributes

Tag Attribute Name Description
colspan Column Span Defines how many columns a 'td' element should span
rowspan Row Span Defines how many rows a 'td' element should span
